CCA Course Notes: Computer Application Basics

Detailed breakdown of computer hardware, peripherals, system architecture, and core software applications.

Central Processing Unit (CPU)

The CPU is considered the **brain of the computer**. It performs all data processing operations, stores data, intermediate results, and instructions, and controls the operation of all other parts of the computer.

Components of the CPU

  • **Memory or Storage Unit:** Stores instructions, data, and intermediate results. Supplies information to other units when needed.
  • **Control Unit (CU):** Responsible for controlling the transfer of data and instructions, managing and coordinating all units, and interpreting instructions. **It does not process or store data.**
  • **ALU (Arithmetic Logic Unit):** Consists of two subsections:
    • **Arithmetic Section:** Performs operations like addition, subtraction, multiplication, and division.
    • **Logic Section:** Performs logic operations such as comparing, selecting, matching, and merging data.

Input Unit and Devices

The Input Unit contains devices used to enter data into the computer. These devices translate information into a form understandable by the computer.

Common Input Devices

  • **Keyboard:** Most popular input device; used for inputting text and data.
  • **Mouse:** Popular pointing device; controls cursor position but cannot enter text.
  • **Joystick:** Pointing device used mainly in Computer Aided Designing (CAD) and computer games.
  • **Scanner:** Works like a photocopy machine to transfer paper-based information to the hard disk for manipulation (Types: Flatbed, Sheet-Fed, Handheld, Drum).
  • **Microphone:** Inputs sound, which is stored in digital form.

Specialized Input Devices

  • **Light Pen:** Pointing device used to select menu items or draw pictures on screen.
  • **Track Ball:** Input device mostly used in notebook or laptop computers instead of a mouse.
  • **Digitizer:** Converts analog information (like signals from a camera) into digital form.
  • **MICR (Magnetic Ink Card Reader):** Used in banks to process cheques printed with magnetic ink.
  • **OCR (Optical Character Reader):** Reads printed text optically, converting characters into machine-readable code.
  • **OMR (Optical Mark Reader):** Used to recognize marks made by pen or pencil, typically for checking multiple-choice answer sheets.

Output Unit and Devices

The Output Unit consists of devices used to get information from the computer. These devices translate the computer's output into a form understandable by users.

Visual Output Devices

  • **Monitors (VDU):** Main output device (Types: CRT - large, high power; Flat Panel - LCD/LED - slim, modern).
  • **Projector:** Uses light and lenses to project a magnified image onto a larger screen or wall.

Printers

Used to print information on paper.

Impact Printers

Print characters by striking them on an ink ribbon pressed onto the paper (Noisy, Low cost, Useful for bulk printing).

  • **Dot Matrix Printer (DMP):** Prints characters as a pattern of dots. (Inexpensive, Slow Speed).
  • **Line Printer:** High-speed printer used with large computer systems for text-based reports (100 to 3800 lines per minute).

Non-Impact Printers

Print without striking a ribbon; print a complete page at a time (Faster, Quieter, High quality).

  • **Laser Printer:** Produces high-quality, fast, and quiet print (up to 200 pages per minute).
  • **Ink-Jet Printer:** Creates images by spraying ink through tiny nozzles onto paper.

Other Output Devices

  • **Plotter:** Special output device designed to produce large drawings or images, such as construction plans or blueprints.
  • **Speaker:** Allows users to listen to sound, music, and voice output.

Memory and Storage Hierarchy

Cache Memory

A very high-speed semiconductor memory that acts as a buffer between the CPU and main memory. It stores parts of data and programs most frequently used by the CPU to speed up processing.

Primary Memory (Main Memory)

Holds only the data and instructions on which the computer is **currently working**. It is **volatile** (data lost when power is off) and faster than secondary memory. A computer cannot run without it.

  • **RAM (Random Access Memory):** Read/write memory.
    • **SRAM (Static RAM):** Data stored in transistors; doesn't need refreshing (used in cache). Low power, faster access.
    • **DRAM (Dynamic RAM):** Data stored in capacitors; requires periodic refreshing (used in main memory). Low cost, greater capacity.
  • **ROM (Read Only Memory):** Non-volatile memory (information stored permanently during manufacture). Stores instructions required to start a computer.
    • Types include PROM, EPROM, and EEPROM.

Secondary Memory (External/Backup)

Used for storing data/information **permanently**. It is **non-volatile** and slower than primary memory.

  • **Examples:** Hard Disk (Local/External), Magnetic Tape, Optical Disks (CD, DVD), Pen Drive, Memory Cards.

Memory Storage Units

**Bit (Binary Digit):** Logical 0 and 1.
**Nibble:** 4 Bits
**Byte:** 8 Bits
**Kilobyte (KB):** $1024$ Bytes
**Megabyte (MB):** $1024$ KB
**Gigabyte (GB):** $1024$ MB
**Terabyte (TB):** $1024$ GB

Software Concepts

Software is the collection of data and computer instructions that tells the computer **how to work**. It is the non-tangible part of the system that makes the hardware perform useful tasks.

System Software

Software designed to **manage and control** the computer's hardware and application programs. It provides the base for all other software to run.

  • **Operating System (OS):** The most important program; acts as an intermediary between the user and the computer hardware (e.g., Windows, macOS, Linux).
  • **Utility Programs:** Assist in maintaining the computer, such as Antivirus, Disk Defragmenter, and Backup tools.
  • **Language Processors (Compilers/Interpreters):** Convert high-level programming code into machine-readable code.

Application Software

Programs designed to perform a specific task or group of tasks for the **end-user**. It helps in performing daily productivity work.

  • **General Purpose:** Used for common tasks (e.g., MS Word, Excel, Web Browsers, Media Players).
  • **Customized/Specific Purpose:** Developed for a particular organization or task (e.g., Railway reservation system, Inventory management software).

Operating System (OS) Basics

The **Operating System** is the primary program that manages all the computer's resources, including hardware and software. It's the environment in which all applications run.

Core Functions of an OS

  • **Memory Management:** Allocates and de-allocates memory space for programs.
  • **Process Management:** Manages the tasks (processes) running on the CPU.
  • **Device Management:** Controls all I/O devices through drivers.
  • **File Management:** Organizes, stores, retrieves, and protects files and directories.
  • **Security:** Protects the system from unauthorized access using passwords and other techniques.

Types of OS (Based on User Interaction)

  • **Single-User OS:** Allows only one user to work on the system at a time (e.g., early versions of Windows, DOS).
  • **Multi-User OS:** Allows multiple users to use the same computer simultaneously (e.g., Linux, Unix, Server editions of Windows).
  • **Multitasking OS:** Allows a single user to run multiple programs or tasks concurrently (e.g., modern Windows and macOS).

File and Folder Management

Effective management of files and folders is crucial for organization and retrieval of information on a computer system.

Files and Folders (Directories)

  • **File:** A collection of related information stored under a single name (e.g., `report.docx`, `photo.jpg`).
  • **Folder (Directory):** A container used to logically group related files and other folders together.

File Naming Conventions and Extensions

  • A file name typically consists of a **Primary Name** and an **Extension**, separated by a dot (`.`).
  • The **Extension** (e.g., `.docx`, `.pdf`, `.mp3`) indicates the file type and the application required to open it.
  • **Common Extensions:**
    • `.docx`: Microsoft Word Document
    • `.xlsx`: Microsoft Excel Spreadsheet
    • `.pptx`: Microsoft PowerPoint Presentation
    • `.pdf`: Portable Document Format (Adobe)
    • `.jpg`, `.png`: Image files
    • `.exe`: Executable program file

Basic File Operations

  1. **Creating:** Making a new file or folder.
  2. **Saving:** Storing a file permanently to a storage device (Hard Disk).
  3. **Copying & Moving:** Duplicating a file or transferring it to a new location.
  4. **Renaming:** Changing the name of an existing file or folder.
  5. **Deleting:** Removing a file or folder from the storage device.
