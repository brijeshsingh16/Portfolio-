Introduction to Computer

A **Computer** is an advanced electronic device that takes raw data as **input** from the user and **processes** it under the control of a set of instructions (called a **program**), gives the result (**output**), and saves it for future use.

Functionalities of a Computer

  1. Takes data as input.
  2. Stores the data/instructions in its memory and uses them when required.
  3. Processes the data and converts it into useful information.
  4. Generates the output.

Computer Generations (1946 - Present)

Generation in computer terminology is a change in the underlying technology being used.

First Generation (1946-1959)

  • Used **Vacuum tube technology**.
  • Unreliable, very costly, and generated a lot of heat.
  • Supported **machine language** only.
  • Huge size, non-portable, and consumed a lot of electricity.
  • Examples: ENIAC, EDVAC, IBM-701, IBM-650, UNIVAC.

Second Generation (1959-1965)

  • Used **Transistors**.
  • More reliable, smaller size, and generated less heat than the first generation.
  • Faster and consumed less electricity.
  • Supported **machine and assembly languages**.
  • Examples: IBM 1620, IBM 7094, CDC 1604, CDC 3600, UNIVAC 1108.

Third Generation (1965-1971)

  • Used **IC (Integrated Circuits)**.
  • More reliable, smaller, faster, and consumed lesser electricity than previous generations.
  • Lesser maintenance and supported **high-level language**.
  • Examples: IBM-360 series, Honeywell-6000 series, PDP (Personal Data Processor), IBM-370/168, TDC-316.

Fourth Generation (1971-1980)

  • Used **Very Large Scale Integrated (VLSI) circuits**.
  • Very cheap, portable, reliable, and very small size.
  • Introduced the concept of **internet** and great developments in networks.
  • Use of PC's and Pipeline processing.
  • Examples: DEC 10, STAR 1000, PDP 11, CRAY-1 (Super Computer), CRAY-X-MP (Super Computer).

Fifth Generation (1980-Till now)

  • Used **ULSI (Ultra Large Scale Integration)** technology.
  • Development of true **artificial intelligence** and **Natural language processing**.
  • Advancement in Parallel Processing and Superconductor technology.
  • More user-friendly interfaces with multimedia features.
  • Examples: Desktop, Laptop, Notebook, Ultra Book, Chrome Book.

Types of Computer

On the basis of working principle

  • **Analog Computer:** Uses continuous physical phenomena such as electrical, mechanical, or hydraulic quantities to model the problem being solved.
  • **Digital Computer:** Performs calculations and logical operations with quantities represented as digits, usually in the binary number system.
  • **Hybrid Computer (Analog + Digital):** A combination of computers capable of inputting and outputting in both digital and analog signals.

On the basis of Size

  • **Super Computer:** The fastest and most expensive type, employed for specialized applications like weather forecasting, fluid dynamics, and nuclear research.
  • **Mainframe Computer:** Very large and expensive, capable of supporting hundreds or thousands of users simultaneously. Supports more simultaneous programs than supercomputers, but supercomputers execute a single program faster.
  • **Mini Computer:** Midsized, multiprocessing system capable of supporting from 4 to about 200 users simultaneously.
  • **Micro Computer (PC):** Includes:
    • **Desktop Computer:** A personal computer sufficient to fit on a desk.
    • **Laptop Computer:** A portable computer with an integrated screen and keyboard.
    • **Palmtop Computer/PDAs:** A hand-sized computer, often without a keyboard, where the screen serves as both input and output.
  • **Workstations:** A terminal or desktop computer in a network; a generic term for a user's machine (client machine) in contrast to a "server" or "mainframe."

Logical Architecture (CPU)

The **CPU** is considered the **brain of the computer**. It performs all types of data processing operations, stores data/instructions, and controls the operation of all parts of the computer. It has three main components:

  1. Memory or Storage Unit
  2. Control Unit (CU)
  3. ALU (Arithmetic Logic Unit)

Control Unit (CU)

Controls the operations of all parts of the computer but does not carry out any actual data processing. It is responsible for controlling the transfer of data and instructions, managing all units, interpreting instructions, and directing the computer's operation.

ALU (Arithmetic Logic Unit)

  • **Arithmetic Section:** Performs arithmetic operations (addition, subtraction, multiplication, division).
  • **Logic Section:** Performs logic operations (comparing, selecting, matching, and merging of data).

Input and Output Units

Input Unit

Contains devices for entering data into the computer, creating a link between the user and the computer by translating information into a computer-understandable form.

  • Keyboard
  • Mouse (most popular pointing device)
  • Joy Stick (used in CAD and gaming)
  • Light pen
  • Track Ball (used in notebooks/laptops)
  • Scanner (Flatbed, Sheet-Fed, Handheld, Drum)
  • Digitizer (converts analog to digital)
  • Microphone (inputs sound)
  • Magnetic Ink Card Reader (**MICR**) - used in banks for cheque processing.
  • Optical Character Reader (**OCR**) - reads printed text character by character.
  • Bar Code Reader (reads data in form of light and dark lines).
  • Optical Mark Reader (**OMR**) - recognizes marks for checking multiple choice answer sheets.

Output Unit

Consists of devices for receiving information from the computer, translating the computer's output into a user-understandable form.

  • **Monitors (VDU):**
    • Cathode-Ray Tube (**CRT**): Large size, high power consumption.
    • Flat- Panel Display (**LED/LCD**): Liquid Crystal/Light-Emitting Diodes.
  • **Printers:** Used to print information on paper.
    • **Impact Printers:** Strike the ribbon onto the paper (Dot Matrix, Line Printers). Very noisy, low consumable costs, useful for bulk printing.
    • **Non-Impact Printers:** Print without using a ribbon (Laser, Inkjet). Faster, quiet, high quality, print a complete page at a time.
  • **Plotter:** Produces large drawings or images, such as blueprints or construction plans.
  • **Speaker:** Allows listening to sound (music, conversation).
  • **Projector:** Uses light and lenses to project a magnified image onto a larger screen or wall.

Memory and Storage

Memory Types

  1. **Cache Memory:** Very high-speed semiconductor memory that acts as a buffer between the CPU and main memory, holding frequently used data for fast access.
  2. **Primary Memory (Main Memory):** Holds data and instructions the computer is currently working on. Limited capacity, usually volatile (data lost when power is off). Divided into RAM and ROM.
  3. **Secondary Memory:** External, non-volatile memory used for storing data/information permanently. Slower than main memory. (e.g., Hard Disk, Magnetic Tape, Optical Disks, Pen Drive).

RAM (Random Access Memory)

Read/write, volatile memory that stores data until the machine is working.

  • **Static RAM (SRAM):** Stored in transistors, requires constant power, does not need refreshing. Faster access, but high manufacturing cost and lower capacity. Used in cache.
  • **Dynamic RAM (DRAM):** Stored in capacitors, gradually discharges energy, requires periodic refreshing. Lower cost and greater memory capacity, but slower access speed and high power consumption. Used in main memory.

ROM (Read Only Memory)

Non-volatile memory from which we can only read. Information is stored permanently during manufacture (e.g., startup instructions).

  • Types: PROM, EPROM, EEPROM.

Hardware and Software

Hardware

The physical and tangible components of a computer (can be seen and touched).

  • **Examples:** Input devices (keyboard), Output devices (monitor), Secondary storage devices (Hard disk), Internal components (CPU, RAM).

Software

A set of programs (sequences of instructions) designed to perform a well-defined function.

  • **System Software:** Designed to operate, control, and extend the processing capabilities of the computer itself. Serves as the interface between hardware and end users (e.g., Operating System).
  • **Application Software:** Designed to satisfy a particular need of a particular environment (e.g., MS Word, photo editors).

Relationship

Hardware and software are mutually dependent. Software cannot be utilized without supporting hardware, and hardware without software is useless. A software acts as an interface between the user and the hardware.

Computer Languages

Communication with computers is carried out through a language bound by rules known as **SYNTAX**.

Low Level Language

  • **Machine Language:** The language understood directly by the computer (0’s and 1’s). Machine dependent and difficult to write/learn.
  • **Assembly Language:** Machine codes are substituted by symbolic codes (**mnemonics**) to improve understanding. First step to improve programming structure.

High Level Language

Uses normal English-like, easy-to-understand statements. Computer independent, making programming easier and simpler.

  • **Examples:** BASIC, COBOL, FORTRAN, C, C++, Java, etc.

Data Representation & Files

Types of Number System

  • Binary Number System (0-1)
  • Decimal Number System (0-9)
  • Octal Number System (0-7)
  • Hexadecimal Number System (0-9 A-F)

ASCII Number

American Standard Code for Information Interchange. Standard that assigns letters, numbers, and other characters to 8-bit code slots (0-255).

  • A-Z (65-90)
  • a-z (97-122)
  • 0-9 (48-57)
  • Space (32)
